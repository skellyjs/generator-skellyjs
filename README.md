# generator-skellyjs
Scaffolding generator for skellyjs

See [skellyjs](https://npmjs.com/package/skellyjs) for information on the framework.

# Installation

```bash
$ npm install -g generator-skellyjs
```

## Create the app:

```bash
$ skelly /tmp/project && cd /tmp/project
```

Install dependencies:

```bash
$ npm install
```

Start the app

```bash
$ npm start
```
